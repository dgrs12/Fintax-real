# Fintax 🐷 - Tu Billetera Inteligente

**Fintax** es una aplicación web de gestión financiera personal diseñada para ayudarte a tomar el control total de tus ingresos y gastos con una interfaz elegante, moderna y de alto impacto visual.

---

## ✨ Características Principales

* **Diseño Crystal UI:** Interfaz basada en *Glassmorphism* (efecto cristal) con fondos oscuros y desenfoques elegantes.
* **Gestión de Usuarios:** Sistema de registro e inicio de sesión seguro.
* **Control de Transacciones:** Registro detallado de ingresos y egresos.
* **Panel Administrativo:** Control total de datos desde la interfaz maestra de Django.
* **Responsive Design:** Optimizado para verse bien tanto en PC como en dispositivos móviles.

---

## 🛠️ Tecnologías Utilizadas

* **Backend:** [Django 6.0+](https://www.djangoproject.com/) (Python)
* **Base de Datos:** SQLite3 (Ligera y rápida)
* **Frontend:** HTML5, CSS3 (Google Fonts: Inter)
* **Entorno:** Virtualenv para gestión de dependencias.
